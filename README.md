# wiki-app

Rick and morty wiki app developed in Next.js, React Query and Typescript

## Deployment app

## Run

```
    npm install
    npm run dev
```

```
    yarn install
    yarn run dev
```

Starts on `http://localhost:3000`

## Build

```
    npm run build
```

## Development technologies and external packages

- Next.js
- Typescript
- React Query 
- Jest & React testing library
- Styled Components
